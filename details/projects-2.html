가상 개인 비서 서비스를 이용한 IoT 스마트 홈 구축

  1. The Implementation of an IoT Smart Home Using a Virtual Personal Assistant Service

Project information

  • Category: Python3, AWS
  • Project date: March, 2018 ~ Jun, 2018
  • Result: 튜터링부문 장려상

Project Details

직전학기 진행한 프로젝트에서 생각났던 아이디어를 추가하여 진행한 프로젝트 입니다.

이번에 새로 추가한 기능은 2가지입니다.

첫번째는 RFID기반 도어락입니다.
현관문 RFID Reader기와 게이트웨이를 조합하여 카드키 인증 시스템을 구현했습니다.

두번째는 사용자 위치 인식 기반 E/V입니다.
집안 내부 E/V를 이용하기위해 근처로 가면 초음파센서가 사용자의 행동을 인식해 E/V를 사용자의 층계로 이동시켜주는 기능을 추가로 탑재했습니다.

게이트웨이는 라즈베리파이3를, 개발언어로는 Python3를, 인공지능 비서는 Amazon의 Alexa를 채택했습니다.

게이트웨이에 연결된 모듈은 도어락에 사용된 RFID R/W기기, E/V 이용에 사용된 초음파센서, E/V를 움직이는데 사용된 DC모터, 집안 온습도 측정 및 제어를 위한 온습도센서, 집안내부 침입자 감지를 위한 PIR센서, 집안내부 침입자 발생시 현장을 촬영하는 카메라 모듈을 사용했습니다.

라즈베리파이 1대의 GPIO로는 전압부족으로 인해 모든 센서를 제어할 수 없어서 2개의 라즈베리파이를 TCP/IP Socket 통신으로 연결하여 사용했습니다.
